ALPHA VERSION

Welcome to microGravity! v0.0.1alpha

Title: “Minimum Viable Product”

Main Features in v0.0.1α

  • File Management: Create, open, save, and save-as functionality
  • PDF Export: Export rendered rich-text markdown documents to PDF
  • Live Preview: Real-time markdown rendering with split-view support
  • Scroll Sync: Editor and preview panes scroll in sync
  • Formatting Toolbar: Bold, italic, headings, lists, quotes, code blocks, links
  • Markdown Support: GFM, LaTeX, GitHub Flavoured syntax highlighting, Mermaid diagrams
  • Session Notes: Temporary overlay that isn’t saved
  • View Modes: Source-only, split-view, and preview-only
  • Theme Support: Light/dark with persistent storage
  • Status Bar: Live word count & cursor tracking
  • Auto-Title Detection: Uses first H1 as filename
  • Missing: UI/UX in its entirety
