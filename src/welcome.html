ALPHA VERSION

Welcome to microGravity! v0.0.1alpha

Title: “Minimum Viable Product”

Main Features in v0.0.1alpha

  • File Management - Create, open, save, and save-as functionality
  • PDF Export - Export rendered, rich-text markdown documents to PDF
  • Live Preview - Real-time markdown rendering with split-view support
  • Scroll Sync - Editor and preview panes scroll in sync
  • Formatting Toolbar - Quick buttons for bold, italic, headings (H1-H3), lists, quotes, code blocks, and links
  • Markdown Support - GitHub Flavored Markdown, LaTeX math equations, syntax highlighting, and Mermaid diagrams
  • Session Notes - Temporary note-taking overlay that doesn’t save to file
  • Three View Modes - Source-only, split-view, and preview-only
  • Theme Support - Light and dark themes with persistent storage
  • Status Bar - Live word count and cursor position tracking
  • Auto-Title Detection - Automatically pulls document title from first H1 heading
  • Missing: UI/UX in any capacity
